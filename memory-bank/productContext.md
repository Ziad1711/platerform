# Product Context

## Why This Project Exists

Moroccan e-commerce businesses struggle with fragmented tools for managing their operations. They typically use:
- Spreadsheets for inventory
- Manual tracking for advertising costs
- Separate systems for orders and suppliers
- No unified view of profitability

**Platerform** solves this by providing an all-in-one solution tailored to the Moroccan market.

## Problems It Solves

### 1. Fragmented Data
- **Problem**: Business data scattered across multiple tools
- **Solution**: Centralized platform with unified data model

### 2. Poor Visibility
- **Problem**: Owners can't see real-time profitability
- **Solution**: Real-time dashboard with key metrics (revenue, costs, profit margins)

### 3. Manual Processes
- **Problem**: Time-consuming manual data entry and calculations
- **Solution**: Automated calculations, integrations with delivery APIs, AI assistance

### 4. Multi-Store Complexity
- **Problem**: Managing multiple stores requires separate systems
- **Solution**: Multi-tenant architecture with store switching

### 5. Language Barrier
- **Problem**: Most SaaS tools are in English
- **Solution**: French interface designed for Moroccan users

## How It Should Work

### User Journey
1. **Onboarding**: User signs up, creates profile with main currency
2. **Store Setup**: Creates one or more stores with specific currencies
3. **Team Management**: Invites team members with appropriate roles
4. **Daily Operations**: 
   - Receives orders (manual entry or API integration)
   - Confirmation agents update order status
   - System tracks inventory movements automatically
   - Delivery companies sync tracking info
5. **Financial Tracking**:
   - Records supplier purchases and payments
   - Logs advertising spend daily
   - Tracks operational expenses
6. **Analysis**:
   - Views dashboard with real-time KPIs
   - Uses AI assistant to ask business questions
   - Generates reports for decision making

### User Experience Goals

- **Speed**: Dashboard loads in < 2 seconds
- **Clarity**: KPIs clearly displayed with visual indicators
- **Simplicity**: Complex operations simplified to essential actions
- **Confidence**: Data accuracy and security always maintained
- **Intelligence**: AI provides insights, not just data
- **Flexibility**: Works for single store or multi-store operations

## Key Differentiators

1. **Moroccan Market Focus**: Currency, language, local delivery integrations
2. **Profitability Focus**: Not just sales tracking, but true profit calculation
3. **AI Integration**: Connected to actual business data for relevant insights
4. **Confirmation Agent Role**: Specific to Moroccan e-commerce workflow
5. **Advertising Cost Allocation**: Per-order ad cost tracking for accurate ROI
